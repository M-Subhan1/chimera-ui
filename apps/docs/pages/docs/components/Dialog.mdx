## Extending Radix Anatomy

### Dialog.Confirm

Chimera adds `Dialog.Confirm`, which is a `Dialog.Close`, except it positions a button in the bottom right corner. You can pass children to it, or use `<Dialog.Confirm />` to use the default button.
